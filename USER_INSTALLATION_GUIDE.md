**A:** Only for initial setup. EDUSCAN works offline for attendance taking.

### Q: Can I change where it's installed?

**A:** Yes! See "Uninstallation" section below.

**A:** Only when taking attendance. You can use EDUSCAN without a webcam for other features.

### Q: Can I install on a USB drive?
**A:** Yes! In Step 7, select your USB drive as the installation folder.

### Q: How much disk space does it take?
**A:** About 500 MB after installation.

### Q: Can multiple users use the same computer?

### Q: What if PATH doesn't work after installation?
