## What Was Wrong
The dashboard runs a Qt event loop. When you clicked "Start Attendance", the code tried to start ANOTHER event loop - which Qt doesn't allow.

## What Was Fixed
**Removed the problematic line:**
```python
sys.exit(app.exec_())  # âŒ Was causing crash
```

Now the code just shows the window and returns to the dashboard's event loop.
